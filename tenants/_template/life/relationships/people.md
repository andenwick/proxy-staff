# People

This file tracks people mentioned in conversations and their relationships.

## Format
Each person has:
- id: Unique identifier
- name: Person's name
- relationship: How they relate to the user
- context: Additional context
- interactions: Log of interactions
- lastContact: When they were last mentioned
