# Boundaries

This file tracks things the AI should never do, always do, and when to escalate.

## Usage
- neverDo: Actions to never take without explicit permission
- alwaysDo: Actions to always take in certain situations
- escalateWhen: Conditions that require human intervention
- limits: Numerical limits (spending, message length, etc.)
